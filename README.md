## Setup
Install [Hugo](https://github.com/gohugoio/hugo) and the [Hugo Pipes requirements](https://gohugo.io/hugo-pipes/postcss/).
```
npm install -g postcss-cli
npm install -g autoprefixer
```

Install node packages.
```
npm install
```
